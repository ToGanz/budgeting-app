# budgeting-app

This is a Single-Page-Application for tracking your finances.
Users can create accounts and budgeting plans. Within these plans they can log their transactions and categorize them.


![image](https://user-images.githubusercontent.com/49613341/125498631-415b38a7-b6c1-42b1-956e-bcddcfaa189e.png)


I built this to get familiar with JWT authentication in a SPA and to learn Tailwind.

The Backend is a Rails API: [https://github.com/ToGanz/budgeting-app-backend]

Technologies used: 
* Vue 3
* Vue CLI
* TailwindCSS



Live on: [https://finance-tracker-38cfd.web.app/]

(Free Plan: first hit takes a few seconds)
